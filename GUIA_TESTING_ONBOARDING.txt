  📊 Verificar CAI en tenant (si lo configuraste):
  
  USE centro_1;  -- Asegúrate de que exista

    • 1 registro con el CAI configurado
    • centro_id = 1

  USE centro_1;
  SELECT * FROM servicios;
  
  ✓ Esperado:
    • 3 registros (o la cantidad que agregaste)

┌──────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                               │
└──────────────────────────────────────────────────────────────────┘

  ❌ Error: "No se puede conectar"
  ✓ Solución: Verificar que el servidor esté corriendo
              php artisan serve

  ❌ Error: "Tenant database does not exist"
  ✓ Solución: Verificar que el tenant se creó correctamente
              php artisan tenants:list
              Si no existe, el sistema lo creará en saveStepTwo()

  ❌ Error: "No redirige al onboarding"
  ✓ Solución: Verificar que onboarding_completed_at sea NULL
              UPDATE centros_medicos SET onboarding_completed_at = NULL
              WHERE id = 1;

  ❌ Error: "Widget no aparece"
  ✓ Solución: Verificar que completaste el onboarding hace menos de 7 días
              El widget solo se muestra durante 7 días

  ❌ Error: 404 en rutas de onboarding
  ✓ Solución: Limpiar caché de rutas
              php artisan route:clear
              php artisan config:clear
