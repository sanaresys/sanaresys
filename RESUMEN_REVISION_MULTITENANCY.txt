## 🎯 PATRÓN DE CORRECCIÓN

### ❌ Código Incorrecto (Antes)
```php
// Filtro innecesario en contexto tenant
$query->where('centro_id', session('current_centro_id'));

// Asignación manual
$data['centro_id'] = auth()->user()->centro_id;

// Campo en formulario
Select::make('centro_id')
    ->default(session('current_centro_id'))
```

### ✅ Código Correcto (Después)
```php
// Sin filtro - el tenant ya filtra
$query->get();

// Sin asignación - no hay columna centro_id
// (eliminar línea)

// Campo eliminado de formulario
// (eliminar campo completo)
```
